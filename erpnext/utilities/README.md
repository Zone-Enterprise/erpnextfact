## Database Migration Utilities

The `migration.py` module provides comprehensive tools for database migrations:

- **Batch Processing**: Process large datasets efficiently in manageable chunks
- **Bulk Updates**: Safely update multiple fields with automatic batching and validation
- **Data Migration**: Migrate data between doctypes with field mapping and transformation
- **Schema Management**: Safely rename fields and manage database indexes
- **Data Validation**: Validate data before and after migrations
- **Progress Tracking**: Real-time progress updates for long-running operations

See [MIGRATION.md](./MIGRATION.md) for detailed documentation and examples.
